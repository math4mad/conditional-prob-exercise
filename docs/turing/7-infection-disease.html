7-infectious-disease

Author

math4mad

1. question

infectious disease 实际和硬币试验原理一样,假设有对某个地区的 20人 做诊断, 来检测该地区的传染病感染比例 在短期内,模型会以一个固定的参数来产生数据 类似以某个概率来扔硬币, 对于受检测的居民,检测结果正面(true)为被传染, 反面(false)没有传染 所有模型构建和硬币的贝叶斯方法相同

2.2 define turing model

Code 
@model function infection(; N::Int)
    # Our prior belief about the probability of heads in a coin toss.
    p ~ Beta(1, 1)

    # Heads or tails of a coin are drawn from `N` independent and identically
    # distributed Bernoulli distributions with success rate `p`.
    y ~ filldist(Bernoulli(p), N)

    return y
end;
